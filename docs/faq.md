# FAQ & Troubleshooting

## Common Issues

### Q: I get a `FileNotFoundError` for the data file.

A: Make sure `data/kplc_interruption_data.json` exists. Download from Kaggle if needed.

### Q: My environment fails to build or dependencies are missing.

A: Use the provided `environment.yml` (conda) or `requirements.txt` (pip) to install all dependencies.

### Q: The dashboard or visualizations are empty.

A: Ensure your data is correctly preprocessed and features are generated before running visualization scripts.

### Q: How do I update the dashboard on GitHub Pages?

A: Regenerate `docs/dashboard.html` and push to `main`. GitHub Pages will update automatically.
